### 上手指南

你需要在你的设备上安装**DevEco Studio**，并且配置相应的**Node.js**环境。该项目的**OpenHarmonySDK**版本已更新至**API18**，项目入口文件位于**entry/src/main/ets/pages/Index.ets**。

#### 安装

克隆本仓库
```sh
git clone https://github.com/langwudong/browser.git
```
